# FUTDERJ

Códigos do futebol de robôs VSSS (Very Small Size Soccer).

## dependências
- protobuf-compiler
- vss-vision

### python
- pyserial
- protobuf
- pygame

## como rodar

compile o protobuf:
```
cd pb
./compile.sh
```

rode o programa:
```
cd ..
python3 main.py [y/b] # dependendo da cor do time
```
