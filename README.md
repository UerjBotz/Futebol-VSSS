# FUTDERJ

Códigos do futebol de robôs VSSS (Very Small Size Soccer).

## dependências
- protobuf-compiler
- vss-vision

### python
- pyserial
- protobuf
- pygame

## como rodar

compile o protobuf (só é necessário na primeira vez):
```
cd pb
./compile.sh
cd ..
```

rode o programa:
```
python3 main.py [y/b] # dependendo da cor do time
```
